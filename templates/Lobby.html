Створення предмета

Структура предмета

Основна частина (обовʼязкова)
  • ID — авто-ідентифікатор предмета (для видачі по ID).
  • name — базова назва від адміністратора.
  • description — базовий опис від адміністратора.
  • image — зображення предмета.
  • h/w — висота та ширина в тетрис-інвентарі.
  • weight — вага однієї одиниці.
  • MAXstr — максимальна міцність предмета.
  • type — тип (береться з таблиці типів).
  • quality — якість (звичайний, незвичайний, епічний, легендарний, міфічний).
Персональна частина (на користувача)
  • remain — кому належить предмет (власник).
  • remain(username) — клітинка інвентаря, де лежить предмет.
  • str(username) — поточна міцність.
  • amount(username) — кількість (1..MAXamount).
  • description(username) — нотатки гравця (не перекриває базовий опис).
  • name(username) — особиста назва (перекриває базову лише для цього гравця).
Таблиця типів
  • weapon
  • ammo
  • armor
  • map
  • food
  • other
